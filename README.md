# ImageModels


## How to Run frontend  

```
cd frontend
npm i
npm run start
```
Open http://localhost:3000/ in browser   
## How to Run backend    
```
cd backend
python -m pip install -r requirements.txt
python main.py

```
creates a backend rest api at port 5000
this is used to interact with the upload page in react
```
